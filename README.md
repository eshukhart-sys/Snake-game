# Snake Garden

Это простая HTML/CSS/JavaScript игра на Canvas.

## Как опубликовать и отправить ссылку друзьям

### Вариант 1: GitHub Pages
1. Создай репозиторий на GitHub.
2. Скопируй в него файлы `index.html`, `style.css`, `script.js`.
3. Закоммить и запушь на ветку `main`.
4. В настройках репозитория включи GitHub Pages для ветки `main` и папки `root`.
5. Через несколько минут ссылка будет доступна в формате:
   `https://<твой-логин>.github.io/<имя-репозитория>/`

### Вариант 2: Netlify / Vercel
1. Подключи репозиторий к Netlify или Vercel.
2. Выбери папку проекта как корень сайта.
3. Сервис автоматически развернёт сайт и выдаст публичную ссылку.

### Вариант 3: Быстрый локальный просмотр
Если хочешь просто протестировать локально, можно открыть `index.html` в браузере.

> Для того чтобы друг играл через ссылку, нужно разместить сайт в интернете: GitHub Pages, Netlify, Vercel или другой хостинг.
